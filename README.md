# Analyse des facteurs influençant la qualité du sommeil

## Description
Ce projet de data science explore les liens entre les habitudes de vie quotidiennes et la qualité du sommeil.

L’objectif est d’identifier les facteurs influençant le sommeil afin de mieux comprendre les comportements pouvant améliorer le bien-être.


## Objectifs
- Analyser les données liées au mode de vie
- Identifier les facteurs impactant la qualité du sommeil
- Proposer des recommandations basées sur les résultats


## Méthodologie

### Analyse exploratoire
- Analyse descriptive des variables
- Visualisation des distributions et tendances

### Analyse statistique et modélisation
- Analyse en Composantes Principales (ACP)
- Modèles de régression
- Test du χ² pour les variables qualitatives


## Résultats
- Identification des variables influentes sur le sommeil
- Mise en évidence de corrélations significatives
- Recommandations sur les habitudes de vie


## Technologies utilisées
- Python
- Jupyter Notebook
- Pandas, NumPy
- Matplotlib
- Scikit-learn
